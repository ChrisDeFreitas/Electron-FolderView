Application Functions
-------------------------
Key          | Function
-------------|-----------
-            | Load last folder viewed
Backspace    | Folder up
F2           | System Info
F3           | Windows Explorer
F4           | Export List
F5           | Reload grid items from disk
F6           | Change Folder
F7           | Bulk Copy/Delete/Move
F8/n         | Create new folder
F9           | Folder size
F11          | Fullscreen
F12          | Dev Tools
Ctrl+F       | Find Item dialog
Ctrl+Shift+I | Dev Tools

Grid Navigation
------------------------------
Key               | Function
------------------|-----------
Up Arrow          | Scroll up
Down  Arrow       | Scroll down
Home              | Scroll to start of grid
End               | Scroll to end of grid
PageUp            | Scroll grid up one screen
PageDown          | Scroll grid down one screen
Alt+S             | Toggle grid auto-scroll

Grid Item Layout
------------------------------
Key               | Function
------------------|-----------
Alt+C             | col mode: arrange items in three columns (default)
Alt+R             | row mode: arrange items in rows
Alt+V             | vert mode: arrange items in a single column with item details
Alt+W             | wall mode: arrange items like posters on a wall
Alt+1 (through 9) | Toggle display of detected file extensions
Alt+H             | Toggle hidden items
Alt+Shift+F       | Toggle folder display through: "Default", "First", "Hidden", "Last"
Alt+Shift+S       | Shuffle items: arrange items randomly

Item Functions
------------------------------
Key               | Function
------------------|-----------
C                 | Copy item or selectList
M                 | Move item or selectList
Delete            | Delete item or selectList
Enter             | Open slideshow with item currently selected (red border)
Alt+Up    Arrow   | Increase display size by .1
Alt+Down  Arrow   | Decrease display size by .1
Alt+Left  Arrow   | Set display scale to .3
Alt+Right Arrow   | Set display scale to 1
Left Click        | Empty selectList and perform default click action for item
Ctrl+Left Click   | Toggle item in selectList (green border)
Alt+A / Ctrl+A    | If selectList empty and itemselected=null then select all items, else empty selectList and/or unselect item
Drop on folder    | Open BulkOperations dialog with selected item or selectList

Slideshow Navigation
------------------------------
Key               | Function
------------------|-----------
C                 | Copy item
M                 | Move item
Delete            | Delete current item
Enter             | Toggle start/close slideshow
Esc               | Close slideshow
Right Arrow       | Next image
Left Arrow        | Prior image
Ctrl+Up    Arrow  | Increase image size by .1
Ctrl+Down  Arrow  | Decrease image size by .1
Right Click       | Context menu
